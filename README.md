1️⃣ What is the difference between var, let, and const?

Answer:
var -> Old way to declare variables.
let -> Modern way to declare variables, and the value can be changed.
const -> Used to declare variables whose value can not be changed.



2️⃣ What is the spread operator (...)?

Answer: The spread operator is use to expand or copy elements of an array or object.



3️⃣ What is the difference between map(), filter(), and forEach()?

Answer: map() -> Creats a new array by modifying each element.
filter() -> Creats a new array with elements that match a condition.



4️⃣ What is an arrow function?

Answer: An arrow function is a shorter way to write a function.



5️⃣ What are template literals?

Answer: Template literals are string with backticks (``) that allow to insert variables inside a string.
